# Transgalactic

Transgalactic is an extension for assembly. Basically saving you some time with writing it. Transgalactic goes through the code and translates its function like syntax into raw assembly, which you can then compile as you wish.

For more information check out the [documentation](https://xantus10.github.io/Transgalactic/)

## Features

- Shorthands for syscall calls
- Advanced functions like strlen and int-str conversion
- Basic loops

## Support

Only compatible with 64bit NASM and Linux syscalls
